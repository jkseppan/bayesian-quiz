Continuous Ranked Probability Score

Lower CRPS = better prediction.

Answer = 42 Confident σ = 1 Uncertain σ = 20
Accurate μ = 42 0.23 4.67

From CRPS to points

Points range from 0 to 100. Higher is better.

points =

100 · e−CRPS / scale

where scale is set per question

100

Perfect prediction (CRPS → 0)

37

Off by one scale unit

0

Wildly wrong (CRPS → ∞)
